# DS2 logo assets

Exported from the live wordmark (`apps/ds-site`). Every file has a **transparent background**.
Each variant exists as both a scalable **SVG** (`svg/`) and a 2276×570 **PNG** (`png/`).

## Variants

| File | What it is | Best on |
| --- | --- | --- |
| `ds2-wordmark-black` | Solid near-black (#15161B) fill | light backgrounds |
| `ds2-wordmark-white` | Solid white fill | dark backgrounds |
| `ds2-wordmark-purple` | Solid brand purple (#6D5DD3) | light/neutral backgrounds |
| `ds2-wordmark-purple-gradient` | Vertical purple gradient (#B4A4F4 → #6D5DD3 → #3A2B86) — the hero treatment | light backgrounds |
| `ds2-wordmark-outline-black` | Black stroke, no fill | light backgrounds |
| `ds2-wordmark-outline-purple` | Purple stroke, no fill | light/neutral backgrounds |
| `ds2-wordmark-outline-white` | White stroke, no fill | dark backgrounds |

> The "DS2" wordmark is **custom-drawn vector art** (not set in a typeface), so it scales cleanly to any size.
> Prefer the **SVG** for print, large canvas, or recolouring; the PNG is for quick drops where SVG isn't supported.

## Brand colours

- **Purple (primary):** `#6D5DD3`
- Gradient stops: `#B4A4F4` (top) · `#6D5DD3` (mid) · `#3A2B86` (bottom)
- Near-black: `#15161B`

## Fonts used on the website

| Role | Typeface | Source | Weights |
| --- | --- | --- | --- |
| **Body / UI text** | **Inter** | Google Fonts | 300, 400, 500, 600, 700 |
| **Display / accent** (the "DIGITAL SOLUTIONS · consulting" tagline, section eyebrows) | **Orbitron** | Google Fonts | 500, 600, 700 |
| Code-style labels, chips, metadata | system monospace stack (`SF Mono` / `JetBrains Mono` / `Menlo`) | system | — |

Both Inter and Orbitron are free, open-source Google Fonts:
- Inter — https://fonts.google.com/specimen/Inter
- Orbitron — https://fonts.google.com/specimen/Orbitron

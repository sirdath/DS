# DS2 research library

A curated, opinionated knowledge base for building **DS2's identity** and **marketing it like a firm that gets paid premium**. Not a link-dump, each entry has *why it matters* for DS2 and *how to apply it*.

Treat this as a living index. Add to it whenever you find a source you'd revisit.

---

## How to use this library

- **Making a positioning / wedge call?** Read 01 + 02 + 03.
- **Making a marketing-asset call?** Read 02 + 04.
- **Writing copy for the site / pitch deck?** Read 01 (foundations) + the signature sentences in POSITIONING.md.
- **Hiring or briefing a designer?** Hand them 01 + 04.

## Conventions

- Every source listed includes **author / book / year + a one-line "why DS2 cares"**.
- Frameworks are summarized in plain words, not jargon.
- "Apply to DS2" sections at the bottom of each doc are the operational takeaways.
- Verified against current sources (May 2026 web pass) where claims are time-sensitive.
